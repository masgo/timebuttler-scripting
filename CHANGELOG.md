## [0.8] - 2026-03-18
### Added
- Adapted the script to the new TimeButler website/UI (Topbar `#topbar-actions` layout).
- Custom topbar links (e.g. Mitarbeiter/Details/Profil), depending on the current user context.
- Repoints the "Gehaltsdaten" link/button to the calendar view when possible.
- Automatically focuses the search input on the Mitarbeiter page.
- Best-effort domain switch support for `timebutler.de` to `app.timebutler.com`.
- Adds a "Kalender" link near the Urlaubskonto/Details area when on the user page.
